# Hover Effects


```postcss
/*
   UTILITIES
   Media Query Extensions:
     -ns = not-small
     -m  = medium
     -l  = large
*/

/* Equivalent to .overflow-y-scroll */
.overflow-container {
  overflow-y: scroll;
}

.center {
  margin-right: auto;
  margin-left: auto;
}
```
