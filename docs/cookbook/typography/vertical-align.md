# Vertical Align

VERTICAL ALIGN
Use vertical align to set typography just right. Vertical align works on inline-level elements (display inline and inline-block) and on table cells.

This documents how vertical align affects text elements. If youâ€™d like to see how to vertically align elements - checkout the layout documentation.

```css
/*

   VERTICAL ALIGN

*/

.v-base     { vertical-align: baseline; }
.v-mid      { vertical-align: middle; }
.v-top      { vertical-align: top; }
.v-btm      { vertical-align: bottom; }

```
